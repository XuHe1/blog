Posted on 2018-09-27

需要对JavaEE规范有较为完整的认识

  • 我们知道session在client是以JSESSIONID放入浏览器cookie中的

  • session的产生

    通常说一次连接建立，默认服务器端就产生了一个会话，但事实上我们如果不手动调用 session创建代码，浏览器端并没有显示cookie，更看不到JSESSIONID，浏览器不显示cookie，cookie不是浏览器的东西吗？
